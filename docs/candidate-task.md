# Candidate Git Task

Complete the steps below using Git and GitHub.

## Task

1. Create a new branch from `main` named `feature/update-team`.
2. In [`TEAM.md`](../TEAM.md), add your first name as a new bullet under `Current Team`.
3. Create or update [`docs/candidate-notes.md`](candidate-notes.md) with:
   - your full name
   - the branch name you used
   - a one-line note describing the change you made
4. Commit your changes with a clear commit message.
5. Push the branch to GitHub.
6. Open a pull request into `main`.
7. Merge the pull request into `main`.

## What this tests

- creating a branch
- editing tracked files
- creating a new file
- staging and committing changes
- pushing a branch
- opening and merging a PR
